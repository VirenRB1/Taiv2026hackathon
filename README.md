# TAIV Hackathon 2026

Think of this as your startup in a box. The repo holds several tracks in separate folders. Move through them at whatever pace works for you—one track end to end, a mix of several, or the full set.

Fork this repo to get started. Each folder has its own README with setup and instructions. Open any of them and go from there.

**Installing Node.js**

You’ll need Node.js (and npm) for the code tracks. Pick your OS:

- **Windows** — Download the LTS installer from [nodejs.org](https://nodejs.org) and run it. Or use Winget: `winget install OpenJS.NodeJS.LTS`
- **macOS** — Download the LTS installer from [nodejs.org](https://nodejs.org), or install with Homebrew: `brew install node`
- **Linux** — Use your package manager (e.g. `sudo apt install nodejs npm` on Ubuntu/Debian, `sudo dnf install nodejs npm` on Fedora) or download from [nodejs.org](https://nodejs.org)

Check that it worked: `node --version` and `npm --version` should print version numbers.
